# Diseño de producto

## Escena

Una administradora trabaja durante el día en recepción, alternando entre computadora y tablet mientras atiende alumnos, confirma transferencias y organiza clases. La interfaz debe ser clara con luz ambiente, rápida y confiable.

## Dirección

- Tema claro por defecto.
- Neutros levemente azulados con celeste argentino como identidad inicial.
- El color de marca puede personalizarse por academia en una etapa posterior.
- Tipografía de sistema para la aplicación.
- Navegación lateral predecible y responsive.
- Densidad media: tablas para operación, espacios generosos para configuración.
- Estados visibles para pendiente, confirmado, vencido, revertido y desactivado.
- Sin gradientes de texto, glassmorphism ni decoración de dashboard genérico.

## Componentes base

- Sidebar derivado del catálogo de módulos.
- Encabezados con contexto y acción principal.
- Tablas con búsqueda, filtros y estados vacíos instructivos.
- Formularios en página o panel lateral; modales solo para confirmaciones breves.
- Alertas con acción concreta y lenguaje no técnico.
- Wizard inicial con resumen de dependencias antes de confirmar.

## Accesibilidad

- Contraste AA.
- Foco visible.
- Controles con etiquetas.
- Navegación por teclado.
- No depender solo del color para comunicar estados.
